### Route: `/admin/courses/new` or `/admin/courses/[id]/edit`

Course editor with module and lesson management.

### Components Used
- `MetricCard` - Stats cards
- `Table` - Course list with actions
- `Button` - Create/Edit/Delete
- `Form` - Course editor form
- `Dialog` - Confirmation dialogs

---

## Data Models Summary

```typescript
interface Course {
  id: string;
  title: string;
  description: string;
  thumbnail: string;
  category: 'engineering' | 'non-engineering';
  price: number;
  modules: Module[];
}

interface Module {
  id: string;
  title: string;
  order: number;
  lessons: Lesson[];
}

interface Lesson {
  id: string;
  title: string;
  duration: string;
  type: 'video' | 'pdf' | 'text' | 'quiz';
  mdxPath?: string;
}

interface UserProgress {
  courseId: string;
  completedLessons: string[];
  percentComplete: number;
}
```
