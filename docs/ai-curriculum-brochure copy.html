● Builders Track
03 / 04
TRACK 1

Builders Curriculum

A comprehensive, implementation-focused journey through the modern AI stack. Each module builds on the previous, creating a complete mental model.

Module 01

Foundations of Regression

Linear & logistic regression from first principles, bias-variance tradeoffs, regularization, and real-world applications. Derive the math, implement from scratch.

Module 02

Mastering Deep Neural Networks

Build fully-connected networks from scratch, derive backpropagation mathematically, diagnose training failures, and understand optimization landscapes.

Module 03

Deep Computer Vision (CNNs)

Convolutional networks, receptive fields, spatial reasoning beyond classification. Understand how CNNs "see" and build intuition for feature hierarchies.

Module 04

Deep Sequence Modelling

RNNs, LSTMs, attention mechanisms, and why transformers dominate long-range dependencies. Build sequence models that actually work.

Module 05

Build Your Own Mini-GPT

Decoder-only transformers, tokenization, causal self-attention, training loop, and text generation. Implement a working language model from scratch.

Module 06

Agentic AI & Multi-Agent Systems

Tool use, memory, planning, coordination, and failure modes of agents. Build systems that can reason and act autonomously.

Module 07

Tree-Based ML Algorithms

Decision trees, random forests, gradient boosting, ensembling mechanics. When and why tree methods outperform neural networks.
